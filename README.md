# 🔔 ContestBell — Frontend
> Never miss a competitive programming contest again.

The frontend for ContestBell — a clean, fast, mobile-friendly web interface 
built with pure HTML, CSS and JavaScript. No frameworks, no dependencies.

→ **Live at: https://contestbell.netlify.app**

## 📸 Pages
- 🏠 **Landing page** — hero, features, how it works, platforms
- 🔐 **Login / Register** — JWT auth
- 📊 **Dashboard** — upcoming contests with platform filters
- ⚙️ **Subscriptions** — manage platforms and divisions
- 👤 **Profile** — user settings and timezone

## ✨ Features
- 📱 Fully responsive — works on mobile and desktop
- 🌍 Shows contest times in user's local timezone automatically
- 🔍 Filter contests by platform
- ⚡ Fast loader animation
- 🎨 Dark theme with yellow accent

## 🛠️ Tech Stack
- **HTML5, CSS3, Vanilla JavaScript** — no frameworks
- **JWT** — stored in localStorage for auth
- **Deployment** — Netlify (auto deploy from GitHub)

## 🔗 Backend
This frontend connects to the ContestBell Spring Boot backend.
